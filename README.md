BatteryMenu

##BatteryMenu is a lightweight native macOS battery monitoring utility built with Swift. It runs directly in the macOS menu bar and provides live battery information in a clean native popup interface.

Designed to stay minimal, fast, and visually consistent with macOS.

⸻

##Features
	•	Live battery percentage in menu bar
	•	Charging and discharging detection
	•	Remaining battery time estimation
	•	Native menu bar popup window
	•	Lightweight background execution
	•	Low memory footprint
	•	Native macOS interface using AppKit

## Installation

### Download Release

Download latest release ZIP from the Releases section and open:

BatteryMenu.app

---

### macOS Security Note

Because the app is unsigned, macOS may block first launch.

To open:

- Right click BatteryMenu.app  
- Select Open  
- Confirm launch
